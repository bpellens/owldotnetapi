Remarks

The name of the namspace is a string consisting of the namespace prefix prefaced with xmlns:. The one exception is xml:base where the value of the base URI of the OWL document is stored under the name xml:base.

Exceptions

Exception Type Condition
ArgumentException Attempt to set a namespace with an empty name.
ArgumentNullException Attempt to store a null namespace.
ArgumentNullException Attempt to set a namespace with a null name.
