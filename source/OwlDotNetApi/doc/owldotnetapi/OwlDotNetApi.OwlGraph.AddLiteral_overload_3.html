IOwlGraph.AddLiteral

Remarks

This method looks in the graph to determine whether a literal with the specified value (and a null datatype and langID) exists in the Graph. If the literal exists a reference to the existing literal is returned. If it does not exist then a new literal (with the specified value, and null datatype and LangID) is created, added to the graph and returned.
